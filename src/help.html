FAQS

What is Off the Edge Coloring?

Off the Edge Coloring is an online coloring book that takes coloring to the next level.

Can I save my coloring pages?

Yes, Off the Edge Coloring allows you to save and export your completed coloring pages to share with the world. Learn more on how to use this feature.

What if I can't find the color i'm looking for?

Off the Edge Coloring offers a variety of colors with the colorpicker tool. Learn more on how to use this feature.

Where did this idea come from?

This idea was a team collaboration of web designers/developers trying to achieve the aesthetic of an edgy and alternative coloring book.

How To Use

Select a coloring page from the gallery.

Select one of the color swatches located at the bottom of the screen and tap on the area you want to start coloring.

If you can't find the color you're looking for, click the color picker icon. The color picker will allow you to tap and drag to the exact color you're looking for.

If you colored an area of your coloring page that you wish to undo or redo, the undo and redo icons will allow you to go back or foward a step.

When you're finished coloring, you can always save your creation by selecting the save icon.
